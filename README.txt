This is a collection of fubction that can used for the topological description of atomic and molecular materials.

Function _plm in legendre_plm: Calculates the associated Legendre polynomials P_lm(cosθ) for a given angle θ.
Function _ylm in spherical harmonics: Calculates the real form of spherical harmonics Y_lm(θ,φ) for given angles θ and φ.
